# JLPT N5 — Referenz-Vokabular

Zentrale Referenz fuer Claude beim Schreiben von N5-Lektionen. Keine Pflicht,
alle zu nutzen — aber **neue Lektionen sollten sich an diesem Korpus
orientieren**, nicht Exoten erfinden.

Struktur: Themen-Cluster → `[Wort] [Lesung] [DE-Bedeutung]`

Fuer vollstaendige Listen siehe:
- `scripts/mnn_data/` (Minna no Nihongo Kapitel 1-25 = N5)
- In der DB: `SELECT word, reading, meaning_de FROM vocabulary WHERE jlpt_level = 5;`

## Gebrauch in diesem Skill

Beim Schreiben einer N5-Lektion:
1. Waehle Thema → 6-10 Vokabeln aus diesem Cluster
2. Baue Beispielsaetze, die **NUR** diese Vokabeln + Grundpartikeln nutzen
3. Bei Zweifel: Hiragana bevorzugen, Kanji weglassen
4. Ein N5-Anfaenger soll jeden Beispielsatz beim ersten Lesen verstehen koennen

**Was NICHT hier steht (= nicht benutzen in N5):**
- Alles ausserhalb der ~800 N5-Kernwoerter
- Passivkonstruktionen (〜られる)
- Potentialform (〜える)
- Konditional (〜たら)
- Keigo (謙譲語, 尊敬語)
